Open Source Product

dev-skills-gemini

AI-powered development workflows for Gemini CLI. Structured coding, debugging, refactoring, architecture, and implementation workflows — directly from the command line.

View on GitHub ↗

Overview

dev-skills-gemini brings structured software development workflows into Gemini CLI. Instead of repeatedly writing ad hoc prompts, developers can run reusable workflows for implementation, debugging, refactoring, and technical decision-making.

Who it's for

Developers, technical builders, indie hackers, and teams who want more consistent engineering workflows across coding, debugging, system design, and code quality improvements.

Key features

  • Structured development skills for Gemini CLI
  • Reusable commands for coding and engineering workflows
  • Global installation through ~/.gemini
  • Practical support for implementation, debugging, refactoring, and architecture

Why it matters

Most AI coding tools generate snippets. Developers need better workflow structure. dev-skills-gemini helps turn AI from a one-off coding assistant into a repeatable development workflow engine.

Example workflows

  • /implement for building features
  • /debug for finding and fixing issues
  • /refactor for improving code quality
  • /architect for system and technical design

Outcome

Better code quality, clearer technical thinking, and more repeatable engineering workflows across projects.
